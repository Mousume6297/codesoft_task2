# 🚀 Karo Pitch – Landing Page

## 📌 Project Overview

This is a simple and responsive **Landing Page** created using **HTML and CSS**.
The project is part of **CodeSoft Internship (Level 1 - Task 2)**.

The landing page is designed to showcase a startup pitching platform where founders can present their ideas to investors.

---

## ✨ Features

* Responsive design (Mobile + Tablet + Desktop)
* Clean and modern UI
* Sticky navigation bar
* Hero section with call-to-action buttons
* Multiple sections (About, How it Works, Startups, Apply)
* Grid layout using CSS
* Smooth scrolling

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)
